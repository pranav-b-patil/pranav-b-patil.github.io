Research & Technical Projects

Retrieval-Augmented Recipe Generation

Natural Language Processing Pipeline

Jan. - April 2026

LLM Grounding, Vector Databases, Transformers

Architected an end-to-end Retrieval-Augmented Generation (RAG) system to solve the issue of LLMs hallucinating ingredients or ignoring dietary constraints in recipe generation. By grounding the generation in a curated corpus rather than relying solely on parametric memory, the system ensures highly accurate outputs.

The pipeline utilizes the all-MiniLM-L6-v2 transformer model to create dense vector representations of over 100,000 recipes, storing them in a FAISS semantic search index. Using strict prompt templates, retrieved context is fed into the Phi-3-mini LLM to enforce consistent output structure. The entire system is deployed as a high-performance RESTful web service via FastAPI and Uvicorn.

Technologies: Phi-3-mini, Hugging Face Transformers, FAISS, FastAPI, Uvicorn, Python

Advanced Mathematical Reasoning using AI Tools

Selected Topics in Mathematics

Sept. - Dec. 2025

Deep Learning, LLM Evaluation, Lean4 Formal Verification

Led a research initiative to evaluate how modern Large Language Models (LLMs) genuinely reason through Olympiad-level mathematics, bypassing standard benchmark data contamination. Utilizing the MathArena testbed, we audited models like Llama-3 and Gemini against fresh, unseen competition problems.

To address the cost limitations of "Best-of-N" sampling, our team architected a verification pipeline aiming to translate natural language proofs generated by the LLM into machine-checked formal representations using the Lean4 theorem prover. We identified critical LLM failure modes—such as hallucinating non-existent mathematical lemmas—laying the groundwork for a more robust "Deep Think"-style multi-path exploration framework.

Technologies: Llama-3, Gemma, Prompt Engineering, Lean4, Python, MathArena
